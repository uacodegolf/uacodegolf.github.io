What is Code Golf?

In a single sentence: write the shortest, nastiest code possible to win challenges!

As per Wikipedia, code golf is "a type of recreational computer programming competition in which participants strive to achieve the shortest possible source code that implements a certain algorithm." In the early days of computing, code golf started on Usenet boards as a way to establish your skill as a programmer. Now, though, the tradition lives on in the form of friendly competition.

Here's how it works:

Below, there are challenges posted by date. You may use any programming language you wish to complete these challenges. Once the challenge closes, we will score each entry by a couple of categories. One category will always be the length of the entry -- the shortest entry for each challenge will be crowned ultimately victorious. Then, we may announce other winning entries by whatever criteria seem fit for the challenge Ã  la the IOCCC. These winning entries, along with other notable mentions, will be published at the end of each challenge.

At the end of the day, we are all here to have fun and share interesting code. So, don't be afraid to submit your programs and ideas! Whether you're a young beginner or a seasoned hacker, you're welcome to participate in our challenges.
